## How to set up

clone the repo

```bash
git clone https://github.com/AhmadNashaat0/bosta-shipment-tracker.git
```

create a .env file and attach this to it:

```bash
VITE_APP_API_URL= https://tracking.bosta.co
```
install all the dependencies:
 
```bash
pnpm i
```

run the project:
 
```bash
pnpm dev
```
